# Dimension Class

## Overview
The `Dimension` class represents a dimensional object that extends from the `Request` class. It contains various properties to store length, width, and height measurements.

## Properties

| Property Name | Type  | Description             |
|---------------|-------|-------------------------|
| length        | mixed | The length measurement. |
| width         | mixed | The width measurement.  |
| height        | mixed | The height measurement. |

## Constructor
The constructor initializes the `length`, `width`, and `height` properties with the provided values.

## Relationships
This class does not have any relationships with other classes.

## Notes
- This class is part of the `Alsendo\AlsendoWrapper\Api\Ecolet\Model` namespace.
- It extends the `Request` class from the same namespace.
