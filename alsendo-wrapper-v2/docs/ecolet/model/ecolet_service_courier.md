# ServiceCourier Class
## Overview
The class `ServiceCourier` represents a service courier in the Alsendo wrapper. It contains properties that define various aspects of a courier, including its ID, slug, name, and status. 
## Properties

| Property | Type   | Description                              |
|----------|--------|------------------------------------------|
| $id      | int    | The unique identifier of the courier.    |
| $slug    | string | The slug (shortened URL) of the courier. |
| $name    | string | The name of the courier.                 |
| $status  | bool   | The status of the courier.               |
