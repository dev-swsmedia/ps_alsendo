## Class: Service
This is a class that represents service details from the Ecolet API order response.
### Properties

| Property     | Type   | Description                              |
|--------------|--------|------------------------------------------|
| $slug        | string | The slug of the service.                 |
| $fullName    | string | The full name of the service.            |
| $courierSlug | string | The slug of the courier for the service. |
| $courierName | string | The name of the courier for the service. |
