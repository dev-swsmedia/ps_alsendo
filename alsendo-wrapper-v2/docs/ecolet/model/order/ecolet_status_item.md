# StatusItem Class
## Overview
The class represents a status item in the Alsendo wrapper, which is used to track the status of an order. It contains properties such as name, real name, and creation date. `StatusItem`
## Properties

| Property     | Type                 | Description                                         |
|--------------|----------------------|-----------------------------------------------------|
| `$name`      | `?string`            | The name of the status item.                        |
| `$realName`  | `?string`            | The real name of the status item.                   |
| `$createdAt` | `?DateTimeImmutable` | The date and time when the status item was created. |
